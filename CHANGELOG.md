# Changelog

All notable changes to `git-ninja` are documented here.

## [1.2.0] - 2026-04-02

- Added Ollama-based commit message generation.
- Added SCM title integration for selected repositories.
- Added `.git/info/exclude` toggle from editor and explorer context menus.
- Added checkout of a file from another branch with validation and confirmation.
- Reduced diff payload size for commit generation.
- Skip newly created files during commit generation.

## [1.0.1] - 2025-07-30

- Fixed invalid icon packaging metadata.
